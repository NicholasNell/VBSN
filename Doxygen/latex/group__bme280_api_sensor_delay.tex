\hypertarget{bme280_api_bme280_cal_meas_delay}{}\doxysubsubsection{bme280\+\_\+cal\+\_\+meas\+\_\+delay}\label{bme280_api_bme280_cal_meas_delay}
This API is used to calculate the maximum delay in milliseconds required for the temperature/pressure/humidity(which ever are enabled) measurement to complete. The delay depends upon the number of sensors enabled and their oversampling configuration.
